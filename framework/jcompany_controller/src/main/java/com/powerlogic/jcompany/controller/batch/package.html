jCompany 3.0 - Ancestrais e serviços dedicados a lógicas Batch.

Rotinas periódicas temporais ou que demorem mais do que 1 (um) minuto preferencialmente devem ser implementadas como rotinas batch, executadas de forma "não interativa" e em horários "fora do pico" de uso OLTP (on-line). Este pacote possui classes jCompany para apoio a este tipo de lógica.
