jCompany 3.0 - Classes de Filtro

Este é um pacote que possui classes que implementam o padrão Filter da especificação Servlet 2.3, viabilizando cadeias de interceptações de requisições, declaradas no web.xml

As classes do jCompany visam injeção do Encoding apropriado (default deve ser o UTF-8, captura de cliques para jCompany Production - Monitor, Cache no Browser, dentre outros.
