jCompany 3.0 - Classes Administrativas da Camada Controle

Este é um pacote que possui classes de controle para utilitários jCompany como verificador de transações JDBC, uso de memória e configurações do Tomcat, dentre outras. Futuramente, com o crescimento, esta natureza de funções será separada em projeto independente.

As classes aqui declaradas não foram criadas com a intenção de serem especializadas, devendo ser acessadas por desenvolvedores em tempo de desenvolvimento como utilitários no menu Area Técnica, ou reservadas para uso exclusivo do jCompany.
