jCompany 3.0 - Serviços da camada controle.

Este pacote concentra serviços auxiliares tópicos da Camada Controle, definidos como POJOs extensí­veis e que são injetados em classes de Controle via Setters automaticamente (ver Injeção de Dependência). Assim, estão disponí­veis em descendentes de PlcBaseAction através de métodos getServiceXXXXX()
