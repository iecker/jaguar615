jCompany 3.0 - Camada de Modelo - Anotação para Transações

Especificação do Pacote

Este pacote contém Annotations no padrão Java 5 que permitem a declaração de modalidade de fechamento de transações. As transações são abertas automaticamente no primeiro momento em que se captura uma sessão de persistência, mas devem ser fechadas pelo desenvolvedor. O jCompany fecha as transações com "commit" em métodos descendentes de PlcBaseFacadeImpl, por default, através do uso de AOP. Pode-se, no entanto, anotar para alterar-se este fechamento ou para que ele não seja realizado
