jCompany 3.0 - Camada de Modelo - Classes Base de Domí­nio e Serviços de IoC e AOP

Especificação do Pacote

Este pacote do jCompany contém classes base para a criação de classes de domí­nio do tipo Application Service ou Business Objects, além de serviços utilizados por esta camada para Inversão de Controle (Instanciamento pelo jCompany), Injeção de Dependência (registro de instância em construtores) e Aspect-Oriented Programming (transações, logging e profiling).
