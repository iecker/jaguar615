jCompany 3.0 - Camada de Modelo - Façade

Especificação do Pacote

Este pacote do contém classes que implementam o Design Pattern Façade (Fachada), que visa isolar a camada Controle (regras de interface e navegação) da camada Modelo (regras do negócio), de modo a diminuir o acoplamento e aumentar a estabilidade e flexibilidade das camadas.
