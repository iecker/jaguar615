jCompany 3.0 - Camada de persistência - Classes Base

Especificação do Pacote

Este pacote do jCompany contém classes de utilização na camada persistência da arquitetura MVC-2, notoriamente classe ancestral que implementa o Design Pattern Abstract Factory, classes para Inversão de Controle (instanciamento de classes pelo jCompany na camada DAO), Injeção de Dependência e AOP.
