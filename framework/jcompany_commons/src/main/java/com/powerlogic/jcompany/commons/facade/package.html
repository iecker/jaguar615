jCompany 3.0 - Interfaces de Façade (Contratos)

Especificação do Pacote

Este pacote concentra as interfaces de fachada (contratos do jCompany entre Camadas Controle e Modelo), basicamente destinadas a implementar Ciclo de Vida de documentos via MVC. (Note que implementações ficam na camada Modelo - somente interfaces são visí­veis na Camada Controle)
