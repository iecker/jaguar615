jCompany 3.0 - Utilitários comuns a todas as camadas MVC. Datas, Reflexão, Arquivos, VOs, etc.

Este pacote concentra utilitários que têm interesse geral para todas as camadas e não possuem as dependências de containers web (Servlets, Struts, etc.) ou de persistência (JDBC, Hibernate).

Estes utilitários não possuem concepção de serem especializados, portanto sao disponibilizados como Singletons com contrutores privados
