jCompany 3.0 - Camada de Classes Comuns - Raiz

Especificação do Pacote

Este é um projeto que possui classes de uso comum tanto na camada Modelo quanto na de Controle, e que devem ser instaladas dos dois lados, na eventualidade de estarem separadas em duas máquinas.

Este pacote raiz contem os POJOs principais da camada, que usualmente trafegam entre as camadas de Controle e Modelo (VOs, Exceção e Outros).
