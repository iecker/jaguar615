Jaguar
O JAGUAR é uma contribuição da Powerlogic à comunidade de software livre que agora passa a contar com um ferramental e framework de integração gratuitos para desenvolvimento Java EE 6, em licença GPLv2 e GPLv3.
É baseado no jCompany Developer Suite, solução com 7 anos de mercado e que já conta com mais de 1.100 projetos em produção, sendo 180 mil pontos de função desenvolvidos em centenas de clientes, públicos e privados. O código inicialmente disponibilizado para o JAGUAR se baseia no jCompany 6.0 Preview, considerando um “Release Candidate” da versão 6.0 oficial, ainda em andamento.
Dicas Sobre Como Aprender o Jaguar

Aprendizado Conceitual incluindo padrões, arquitetura geral, usabilidade, etc., sem requerer prática (ideal para coordenadores, projetistas e arquitetos):
1. Leia o Módulo A (capítulos de 1 a 5) do livro “Tirando o Máximo do Java EE 6 Open Source”, disponível para download gratuito em Documentação.
2. Em seguida leia os tutoriais do Módulo B, concentrando-se no processo e resultado obtidos.
3. Assista aos vídeos no YouTube em Vídeos.
Aprendizado Prático (ideal para desenvolvedores):
1. Baixe o produto conforme instruções de download.
2. Instale o Jaguar conforme instruções do Capítulo 2 do livro “Tirando o Máximo do Java EE 6 Open Source”, disponível para download gratuito em Documentação.
3. Crie um novo projeto seguindo a prática do Capítulo 3.
4. Em seguida desenvolva atentamente os tutoriais do Módulo B (capítulos 6 a 11).
Importante: pode-se estudar os demais capítulos conforme o interesse específico, porém eles sofrerão atualizações importantes previstas para 31/janeiro/2011, quando a Powerlogic cederá uma nova versão de atualização para o Jaguar, com dezenas de inovações. Portanto, sugere-se complementar o estudo para além dos módulos A e B após este lançamento.
Aprendizado Através de Cursos Oficiais:
Considere registrar-se para algum curso oficial da Powerlogic, fabricante do Jaguar. Para tanto envie um email com interesse para jaguar@powerlogic.com.br.
