JAGUAR
O JAGUAR é uma contribuição da Powerlogic à comunidade de software livre que agora passa a contar com um ferramental e framework de integração gratuitos para desenvolvimento Java EE 6. Por reunir uma série de conceitos e padrões de implementação inovadores, sugerimos que sejam seguidas as dicas abaixo para o seu melhor entendimento.
Dicas sobre como aprender o JAGUAR
Aprendizado Conceitual incluindo padrões, arquitetura geral, usabilidade, sem requerer prática (ideal para coordenadores, projetistas e arquitetos):
Leia o Módulo A (capítulos de 1 a 5) do livro “Tirando o Máximo do Java EE 6 Open Source”, disponível para download gratuito no site do JAGUAR;
Leia os tutoriais do Módulo B, concentrando-se no processo e resultado obtidos;
Assista aos vídeos disponíveis no site do JAGUAR.


Aprendizado Prático (ideal para desenvolvedores):
Desenvolva atentamente os tutoriais do Módulo B (capítulos 6 a 11).

Aprendizado Através de Cursos Oficiais:
Considere registrar-se para algum curso oficial da Powerlogic, fabricante do Jaguar. Para tanto envie um email com interesse para jaguar@powerlogic.com.br.
