Renderiza o formulario com Tab-Folder da paleta jQuery UI, utilizado como elemento de leiaute (template Facelets). Este mecanismo facilita a manipulação genérica do formulário, viabilizando a retirada de abas dinamicamente em tempo de execução, para impressão e conferência, além de controle da tabulação, conforme abaixo definida. Dica: Para fazer seu próprio leiaute de tab-folder ou formulário, altere o template abaixo para /fcls/template/PlcGeralTemplate.xhtml e mova o conteúdo dos XHTML gerados para dentro de ui:define name="plc-corpo-formulario"
